Journey 8: Committee Dashboard

Board Administrator monitors and manages committee health across the organization

User Journey Overview

As a Board Administrator
I want to monitor the health and performance of all committees from a single dashboard
So that I can identify at-risk committees, track engagement metrics, and ensure committees are functioning effectively

Key Dashboard Features

Health Scores

Composite metrics showing overall committee health (0-100) with color-coded indicators

Engagement Metrics

Member participation rates, active/inactive counts, communication frequency

Action Items

Completion rates, overdue counts, average time to completion

Attendance Tracking

Meeting attendance rates, trends, quorum achievement, absence patterns

Risk Indicators

Committees requiring attention, declining trends, governance violations

Reporting & Export

Generate PDF reports, export to CSV/Excel, schedule automated reports

Option A

Card-Based Dashboard

Design Philosophy

Visual, scannable layout with emphasis on health scores and quick identification of issues through color-coded committee cards.

Key Features

  • Committee cards with prominent health scores
  • Color-coded borders for status (red/yellow/green)
  • Mini charts embedded in each card
  • Responsive grid layout
  • Quick action buttons per committee
  • Filter and sort controls at top
Strengths
  • Highly visual and scannable
  • Quick pattern recognition
  • Easy to spot problem committees
  • Intuitive for non-technical users
Limitations
  • Less dense information display
  • Harder to compare metrics
  • More scrolling required
  • Limited sorting capability

Option B

Data Table Dashboard

Design Philosophy

Dense information display with powerful sorting/filtering, optimized for analytical work and metric comparison.

Key Features

  • Comprehensive table with all metrics
  • Advanced sorting and filtering
  • Inline sparklines and mini charts
  • Expandable rows for details
  • Bulk actions and export
  • Column customization
Strengths
  • Maximum information density
  • Powerful comparison tools
  • Efficient for data analysis
  • Quick multi-sort capability
Limitations
  • Can feel overwhelming
  • Less visually engaging
  • Steeper learning curve
  • Harder to present to execs

Design Philosophy

Executive overview with emphasis on trends, patterns, and high-level insights through charts and visualizations.

Key Features

Limitations

  • Charts can be misinterpreted
  • Performance with large datasets
