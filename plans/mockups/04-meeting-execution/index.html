Journey 4: Meeting Execution

Live Meeting Management & Note-Taking

User Journey Context

As a Committee Secretary

I want to conduct a meeting with integrated tools for attendance, agenda management, note-taking, and action tracking

So that I can focus on capturing accurate information without juggling multiple systems or documents

Key Features

Attendance

  • Mark Present/Absent/Excused
  • Track late arrivals
  • Real-time quorum status
  • Proxy tracking

Agenda Control

  • Visual progress indicator
  • Item-by-item timer
  • One-click advancement
  • Skip/reorder on the fly

Note Capture

  • Rich text editor
  • Auto-organized by item
  • Motion templates
  • Auto-save every 30s

Action Items

  • Quick capture in-flow
  • Member assignment
  • Due dates & priorities
  • Linked to agenda items

Voting

  • Record motions
  • Track proposer/seconder
  • Roll call or voice vote
  • Auto-calculate results

Finalization

  • Preview formatted minutes
  • Edit/review mode
  • Export PDF/Word
  • Email distribution

Design Options

Option A: Linear Flow

Step-by-step progression through meeting phases

Sequential Navigation

Strengths

  • Clear mental model of meeting flow
  • Reduced cognitive load (one task at a time)
  • Mobile-friendly sequential layout
  • Easy to learn for new users

Considerations

  • Context switching between sections
  • May slow down experienced users
  • Harder to capture notes while progressing
Try Prototype

Option B: Split-Screen

Agenda on left, workspace on right

Dual-Panel Interface

Strengths

  • Always see agenda context
  • Efficient multi-tasking
  • Dense information layout for power users
  • Quick switching between tasks

Considerations

  • Can feel cramped on smaller screens
  • Multiple panels may overwhelm new users
  • Requires larger display for comfort
Try Prototype

Option C: Presenter Mode

Large display with floating panels

Full-Screen Display

Strengths

  • Optimized for projector/screen sharing
  • Clear focus on current agenda item
  • Minimalist, distraction-free interface
  • Great for hybrid/remote meetings

Considerations

  • Floating panels can obscure content
  • Less efficient for rapid data entry
  • May require larger display
Try Prototype
